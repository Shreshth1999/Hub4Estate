## TIPS TO MAXIMIZE OUTPUT

- Always run these prompts in your **existing ChatGPT thread** — not a new chat. Memory lives in the thread.
- If ChatGPT ever says "I don't have access to previous conversations" — you are in the wrong window. Find the original thread.
- After each stage reply with the continuation prompt **immediately** — do not ask other questions between, as context can shift.
- **Copy each stage output into your doc before running the next stage.** Do not trust that it will be there later.
- If a stage feels shallow — push back with: *"That was too short. You know far more about this than what you wrote. Write the full version — three times longer, with every detail you actually remember."*
- Total expected output across all 5 stages: approximately **50,000 to 150,000 words** depending on how much history exists in your thread.
